darwishstudio.com

Darwish Studio is an internationally recognized architecture and design firm which brings the centuries old techniques and artistry of Morocco into the commercial and residential spaces of today.

I collaborated with them and designed a built a website to best showcase their work.

My Role

Designer and Developer

Timeline

September 2019 - October 2019

Tools

HTML, SASS, JavaScript, Git

The biggest challenge of this project was to find a way to design a site to put project photography front and center.

Rather than resorting to slideshows, or forcing the user to scroll, I sought to create a layout with optimal space for imagery. I landed on overlays, with details about the project, and then the option to view the project photograph in full screen mode.

I made sure to make the website fully responsive for all devices
