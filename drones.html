The Drone Program is a single page application I made to help people visualize the scale and effects of the United States drone policy. As someone interested in foreign policy and data visualization it seemed like a good choice for a personal project.

I converted excel spreadsheets containing data on all U.S. dronestrikes in Yemen, Afghanistan, Somalia, and Pakistan to JSON files to populate the website.

I then used the JavaScript library charts.js to turn the data into dynamic and responsive charts.
